# Find-My-Fountain
Web application to locate drinking water fountains in NYC parks.

Developed by Jackson Tullman, Angeleanne Enriquez, Aidan Cancelliere, and Jacob Barbulescu

This website can be used to find and review fountains throughout New York City. Filter by different metrics such as taste and cleanliness, and find the perfect fountain for you!

Instructions:
Install the codebase and use npm i to install all dependencies.

Run "npm start" to launch server (fountains will be automatically seeded)

Register for an account

Search for fountains, leave reviews, and have fun!
